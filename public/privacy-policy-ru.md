# Политика конфиденциальности Clash

**Последнее обновление:** 1 апреля 2026 г.

## 1. Общие положения

Clash — это открытое программное обеспечение (open source), которое может использоваться для развертывания собственных серверов управления турнирами. Настоящая политика конфиденциальности описывает принципы обработки персональных данных в экосистеме Clash.

**Важно:** Clash распространяется как open source проект. Каждый экземпляр сервера управляется независимым оператором. Настоящая политика является **рекомендательной** и должна быть адаптирована каждым оператором под свои юридические требования.

## 2. Какие данные обрабатываются

### 2.1. Данные, предоставляемые пользователем

- **Учетные данные:** имя, email, пароль (в хешированном виде)
- **Профиль бойца:** имя, псевдоним, категория, количество побед
- **История турниров:** участие, результаты, рейтинги

### 2.2. Автоматически собираемые данные

- **Технические данные:** IP-адрес, версия клиента, тип устройства
- **Данные сессий:** cookies, токены аутентификации
- **Логи активности:** время входа, действия в системе

### 2.3. Блокчейн данные (при интеграции)

- **Публичный адрес** кошелька пользователя
- **Рейтинги бойцов** (публичные данные)

## 3. Правовые основания обработки

Обработка персональных данных осуществляется на следующих основаниях:

- **Исполнение договора** — предоставление функционала турниров
- **Согласие пользователя** — маркетинговые коммуникации
- **Законный интерес** — обеспечение безопасности, предотвращение мошенничества
- **Юридические обязательства** — хранение данных по требованию законодательства

## 4. Цели обработки данных

| Цель                              | Данные                   | Срок хранения           |
| --------------------------------- | ------------------------ | ----------------------- |
| Предоставление доступа к аккаунту | Email, пароль            | До удаления аккаунта    |
| Проведение турниров               | Данные бойца, результаты | Бессрочно (для истории) |
| Расчет рейтинга                   | История матчей           | Бессрочно               |

## 5. Передача данных третьим лицам

### 5.1. Обязательные передачи

- **Блокчейн** — публичные рейтинги передаются в сеть Polygon
- **Хостинг-провайдеры** — данные хранятся на серверах оператора
- **Платежные системы** — при обработке транзакций

### 5.2. Не передаются

- Персональные данные не продаются третьим лицам
- Не передаются рекламным сетям

## 6. Права пользователей

В зависимости от юрисдикции, пользователи имеют право:

- **На доступ** — получить копию своих данных
- **На исправление** — исправить неточные данные
- **На удаление** — удалить аккаунт и данные
- **На ограничение** — ограничить обработку
- **На переносимость** — получить данные в машиночитаемом формате
- **На возражение** — отказаться от обработки

**Для реализации прав** обратитесь к оператору вашего экземпляра Clash.

## 7. Хранение и безопасность

### 7.1. Хранение

- Пароли хранятся в хешированном виде (argon2id)
- Персональные данные хранятся в PostgreSQL
- Данные рейтингов могут быть записаны в блокчейн (необратимо)

### 7.2. Меры безопасности

- Регулярные обновления безопасности
- Бэкапирование данных
- Контроль доступа к серверам

## 8. Передача данных за границу

Данные могут передаваться в страны, где расположены:

- Серверы оператора
- Хостинг-провайдеры
- Блокчейн сеть Polygon (глобально)

Оператор обязан обеспечить уровень защиты не ниже требований законодательства страны пользователя.

## 9. Cookies и трекинг

Clash использует:

- **Session cookies** — для аутентификации

Третьи стороны не используются для трекинга.

## 10. Для операторов серверов

Если вы разворачиваете собственный сервер Clash:

1. **Вы являетесь оператором персональных данных**
2. **Обязаны** уведомить пользователей о своей политике
3. **Рекомендуется** адаптировать данную политику под:
   - Вашу юрисдикцию
   - Фактические настройки сервера
   - Способы обработки данных

4. **Минимальные требования:**
   - Укажите контактные данные для вопросов о данных
   - Опишите сроки хранения
   - Предоставьте способ удаления аккаунта

## 11. Контакты

### Для пользователей

По вопросам персональных данных обращайтесь к оператору вашего экземпляра Clash.

### Для операторов и разработчиков

- GitHub: [https://github.com/Clash-org](https://github.com/Clash-org)
- Email: clash_org@inbox.ru (для вопросов о проекте)

## 12. Изменения в политике

Политика может обновляться. Крупные изменения будут анонсироваться в репозитории проекта.
